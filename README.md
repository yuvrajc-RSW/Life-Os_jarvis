# LifeOS + Jarvis

This project is a comprehensive SaaS platform designed to be a "Calm, Intelligent, Human" personal assistant and family dashboard.

## Project Structure

- **/lifeos-app**: The core web application built with Next.js and TypeScript.
- **/Life-OS**: Design documentation, sitemaps, and AI architecture specifications.
- **CLAUDE.md**: Development guidelines and architectural principles.
- **AGENTS.md**: AI agent configurations and behavioral rules.

## Getting Started

To run the web application:

```bash
cd lifeos-app
npm install
npm run dev
```

## Vision

LifeOS Jarvis aims to provide a proactive, intelligent dashboard that manages expenses, insights, and family activities using natural language interactions (Hinglish supported).
